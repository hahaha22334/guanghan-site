# AGENTS.md

## Project Overview

Single-page static website about Guanghan, Sichuan (四川广汉) - a cultural tourism introduction site featuring Sanxingdui archaeological findings.

## Structure

```
index.html          # Main HTML file with inline CSS/JS (~55KB)
guanghan.html       # Legacy copy (can be removed)
images/             # Original source images
images/optimized/   # Optimized WebP/AVIF images (generated by optimize-images.js)
optimize-images.js  # Sharp-based image optimization script
package.json        # Node.js project with sharp dependency
```

## Key FACEBOOK/           # Cloudflare Pages deployment config (headers, redirects)
```

## Key Facts

- **No build system** - pure HTML/CSS/JS, no framework, no bundler
- **Image optimization**: Run `node optimize-images.js` to generate WebP/AVIF variants at multiple widths (400w, 800w, 1200w, 1920w)
- **External dependencies** (CDN):
  - Google Fonts: Noto Serif SC, Noto Sans SC (loaded with `media="print" onload` for async)
  - Font Awesome 6.4.0 icons (async loaded)
- **Deployment**: Cloudflare Pages (zero-config, auto-deploys from Git)
- **Responsive design**: Mobile-first with media queries at 1024px, 768px, 480px breakpoints
- **Chinese language site** (lang="zh-CN")
- **Color scheme**: Bronze/gold/cream CSS variables (`--bronze`, `--gold`, `--cream`)
- **Animations**: Scroll-triggered fade-in via IntersectionObserver
- **Hero slider**: Auto-advance (5s) + manual controls + touch swipe support

## Development Commands

```bash
# Optimize images (run after adding new source images to images/)
node optimize-images.js

# Local preview - just open index.html in browser
# No server required, but for proper testing use a static server:
npx serve .
# or
python -m http.server 8000
```

## Deployment

**Cloudflare Pages** (recommended, zero-cost):
1. Push to GitHub/GitLab
2. Connect repo in Cloudflare Pages dashboard
3. Build command: `exit 0` (no build step)
4. Output directory: `/` (root)
5. Auto-deploys on push to main branch

**Cloudflare Pages config** (in repo root):
- `_headers` - Cache headers, security headers, Brotli/Gzip
- `_redirects` - SPA fallbacks if needed

## Image Optimization Details

Source images in `images/` → optimized to `images/optimized/`:
- Hero images (轮放1-4): 400w, 800w, 1200w, 1920w
- Content images (房湖公园, 金雁湖, 牛杂火锅, 城市概况): 400w, 800w, 1200w
- Formats: WebP (quality 80) + AVIF (quality 70)
- HTML uses `<picture>` with `srcset` for responsive loading

## Conventions

- Chinese language site (lang="zh-CN")
- CSS custom properties for theming
- All images use local optimized versions with `loading="lazy"` (except hero first slide)
- Touch swipe support on hero slider
- Font Awesome icons via CDN
- No external image hotlinking (all local)
